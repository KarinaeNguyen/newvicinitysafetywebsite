Short Description

A compact remote control that allows you to quickly turn on/off or test the entire fire alarm system within a 50m range.

Detailed Description

The SafeFire HM1RC-W Remote Control offers absolute convenience for system operators. Instead of climbing up or getting close to each sensor to check, test, or reset alarms, you can simply use this handheld remote to control the system within a 50m radius.

The device boasts an ultra-compact design, making it easy to carry or attach to a keychain. The remote utilizes a standard coin cell battery with an extended lifespan of 3 to 5 years. Perfect for site managers and facility operators who need effective system control without constant manual intervention.

Key Features:

  • System test functionality (no alarms triggered)
  • 50m control range in open areas
  • Ultra-compact and portable design

  • Extended battery life: 3-5 years
